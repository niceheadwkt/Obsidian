# Claude Design 與前端美化實務

**Claude Design 與前端美化實務** 探討了 Anthropic 最新推出的 Claude Design 前端 UI/UX 視覺美化引擎。這套工具整合了即時程式碼預覽、互動式微調以及標註留言功能，讓非技術背景的設計師或開發者能夠以直觀、所見即所得的方式，將粗糙的前端網頁或簡報提煉出世界級的品牌設計質感。

---

## 1. Claude Design 核心功能與互動微調機制

Claude Design 的運作邏輯是將 AI 的代碼寫入能力與網頁渲染引擎結合。在您輸入前端代碼或簡報大綱後，右側會彈出獨立的互動視窗，支援以下三種核心微調手段：

### 1. XML Prompt (XML 提示詞結構)
在要求 Claude Design 進行排版與配色時，使用結構化的 XML 標記包裹設計規範，能讓 AI 以極高的依從度執行樣式渲染。例如：
```xml
<design_spec>
  <theme>Minimalist Business</theme>
  <primary_color>#0033aa</primary_color> <!-- 皇家藍 -->
  <secondary_color>#f4f4f4</secondary_color>
  <border_radius>8px</border_radius>
  <layout_grid>3_columns</layout_grid>
</design_spec>
```

### 2. Slider (滑動條數值微調)
在右側預覽介面中，系統提供了視覺化的滑動條 (Sliders)。使用者可以直接用滑鼠左右拖動滑動條，微調元件的**間距 (Padding/Margin)**、**圓角弧度 (Border-radius)**、**字體粗細 (Font-weight)** 或**陰影深度 (Box-shadow)**。調整後，Claude 會自動在左側後台重寫對應的 CSS 樣式代碼。

### 3. Comment (線上標註留言修改)
這是最富革命性的協作方式：
*   **操作**：直接在右側預覽畫面上，用滑鼠拖曳框選需要修改的區域（如：一個偏小的按鈕或是不對齊的表格）。
*   **留言**：在彈出的輸入框中直接打字（如：「請將此按鈕的字體加粗，並將背景顏色換成深灰色」）。
*   **執行**：Claude 會精確定位被框選區域的 HTML/CSS 圖層，並僅針對該局部代碼進行重構，避免整頁重寫。

### 4. 互動視覺功能 (Interactive Visualization)
最新的 **「互動視覺 (Interactive Visualization)」** 功能允許用戶直接在對話框中生成可互動的圖表與視覺看板。AI 能即時處理複雜的數據，並在預覽中渲染出動態的長條圖、折線圖或雷達圖，無需安裝額外的套件（詳見 [[raw/2026-06-15T160047+0800-Claude推「互動視覺」新功能，可以直接在對話框裡畫圖表.md|Claude 互動視覺功能說明]]）。

## 2. 商業美學實踐：麥肯錫風格 (McKinsey Style)

在商業匯報與高端產品提案中，「麥肯錫風」是極受歡迎的極簡商務視覺設計。

### 🎨 麥肯錫風格要素
*   **色彩系統**：純白或極淺米色背景（90% 佔比）、**皇家藍 (Royal Blue)** 作為核心聚焦色、暗灰與香檳金作為輔助與點綴。
*   **排版美學**：精品級的無襯線字體、嚴格的網格系統（Grid System）、強大的對比度與**高達 40% 以上的留白率**，拒絕花哨的過渡裝飾。

### 📝 麥肯錫風 Prompt 範本
當使用 Claude Design 生成簡報或網頁 UI 時，可輸入以下指令：

```markdown
請將此頁面的 UI 排版美化為「麥肯錫商業諮詢風」。請遵循以下視覺指南：
1. 配色：主色使用皇家藍 (#0A2540)，輔色使用優雅的暗銀灰 (#F6F9FC)，背景為純白。
2. 排版：使用現代無襯線字體，行高設為 1.6 倍，整體畫面保持 45% 以上的高留白率，營造呼吸感。
3. 元件：所有的數據卡片與表格，使用微弱的柔和陰影與 6px 圓角邊框，並以嚴格的網格系統 (CSS Grid) 水平與垂直對齊，呈現高端、專業且清晰的商業提案美感。
```

## 3. 搭檔 Refero 進行 Token 節省優化

由於 Claude Design 在渲染與反覆修改視覺元件時需要頻繁讀寫整個 HTML/CSS 程式碼，這會導致對話上下文迅速飆升，極易達到單日用量上限。

### 💡 搭配 Refero 設計庫優化
*   **原理**：**Refero** 是一個收集了全球頂尖 SaaS 與品牌網頁設計組件（如：登入頁面、定價表格、導覽列）的視覺參考庫。
*   **實踐**：
    1.  先在 Refero 上找到您心儀的設計組件。
    2.  將該組件的排版邏輯與配色，精簡成簡短的文字規格（例如：「Refero 類型的左側導覽列，含圓角微凸的 Hover 效果」）。
    3.  僅將此精簡規格提供給 Claude Design 作為引導。
*   **效果**：這能精準定位 AI 的視覺方向，**避免 AI 反覆盲目生成數十次**而燒光 Token。
