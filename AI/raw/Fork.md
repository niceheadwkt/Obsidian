「Fork」這個詞在不同的領域有著完全不同的定義，通常最常見的是以下幾種情境：

- **Git / GitHub 專案複製**：在版本控制中，將別人的專案倉庫（Repository）複製一份到自己的帳號下，讓你可以自由修改且不影響原專案，之後也能透過 Pull Request 貢獻程式碼。
    
- **Unix / Linux 系統呼叫 (`fork()`)**：在作業系統中，一個行程（Process）用來建立一個與自己幾乎完全相同的子行程（Child Process）的系統呼叫。
    
- **西洋棋戰術（雙擊）**：用一個棋子（通常是騎士）同時攻擊對方的兩個或多個重要棋子，逼對方必須做出抉擇。
    
- **日常餐具**：用來吃義大利麵、牛排或沙拉的叉子。
