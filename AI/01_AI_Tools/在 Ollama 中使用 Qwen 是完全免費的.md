Ollama 本身是一個**開源且免費**的本地端 AI 模型執行工具，而 Qwen（通義千問）官方釋出的也是開源權重（Open-weight）的模型。

當你在電腦上使用 Ollama 運行 Qwen 時，整個運作機制是這樣的：

- **沒有訂閱費或 API 呼叫費：** 你不需要註冊帳號、不用綁定信用卡，不論你一天生成多少字、呼叫幾萬次 API，都不會產生任何軟體費用。
    
- **完全本地運行（Local）：** 模型是直接下載到你的硬碟中，並利用你電腦的 **CPU 或顯示卡（GPU）** 來進行計算。所有的資料和對話都不會上傳到雲端，隱私性極高。
    
- **唯一的「代價」是硬體與電費：** 唯一的成本就是你電腦的硬體配置（硬碟空間、記憶體/顯存）以及跑模型時所消耗的電量。
    

### 如何在 Ollama 中免費執行 Qwen？

你只需要打開終端機（Terminal 或 Command Prompt），輸入對應尺寸的指令，Ollama 就會自動下載並運行。

目前 Ollama 上最熱門的是 **Qwen 2.5** 系列，針對不同硬體規格，你可以選擇合適的模型大小：

[[bash|Bash]]

```
# 適合一般筆電、輕度開發、測試（約需 4GB 以上記憶體）
ollama run qwen2.5:3b

# 效能與速度平衡的黃金版本（建議 16GB 記憶體，或有 6GB 以上顯存的獨立顯卡）
ollama run qwen2.5:7b

# 專門用來寫程式、Debug 的程式碼優化版本（70億參數）
ollama run qwen2.5-coder:7b
```

> 💡 **小提示：** 下載完成後，你就可以直接在終端機與它對話。如果你是軟體開發者，Ollama 隨後會在背景自動開啟一個本地的 API 服務（預設是 `http://localhost:11434`），你可以直接用 Python、Node.js 或是將它接入 **Cursor、vLLM、[[Dify]]** 等開發工具中，這一切也都是免費的。
