Groq 是一家專注於 **AI 推理（Inference）加速**的硬體與技術公司。他們最核心的創新是開發了 **LPU（Language Processing Unit，語言處理單元）**。

與傳統的 GPU 或 TPU 不同，LPU 是專門為了處理大語言模型（LLM）等序列化文字生成而設計的，其最大的特點就是**快**——能提供幾近即時、極低延遲的文字生成體驗。

### 核心優勢

- **極致的生成速度：** 能夠以每秒數百個 Token（Tokens per second）的速度輸出文本，比傳統 GPU 雲端服務快上數倍。
    
- **LPU 確定性架構：** 傳統 GPU 依賴複雜的快取與記憶體管理，而 LPU 採用確定性架構，能精準控制資料流，從而最大化運算效率並降低耗能。
    
- **相容性高的 API：** 提供與 OpenAI 格式完全相容的 API 接口，讓開發者可以輕鬆將極速推理能力無縫整合進現有的應用、自動化腳本或 AI Agent 中。
    

### 目前支援的熱門模型

Groq 雲端平台主要運行頂級的開源模型，包括但不限於：

- **Meta Llama** 系列
    
- **Mistral / Mixtral** 系列
    
- **Google Gemma** 系列
